## smahn9123 (안승민)

**증상 카운트 v3 API 신설 (OR-1424)** — `/detect/nodes/count`를 distinct(고유 증상) + total($sum) + lastTime을 **단일 aggregate 파이프라인**으로 한 번에 산출하도록 신설. manager-web을 v2의 dead query 2개에서 v3 단일 호출로 마이그레이션하면서 **dead 위젯 4종(−2,920줄)**을 정리했다.

**노드 인증 테스트 11건 추가 (OR-742)** — `auto_pin_new_node` 설정 동작을 user_key 유무·SettingStore None·manager None·auto_pin true/false 경계까지 모두 커버(+448줄).

**logrotate 운영 개선 (OR-1141)** — mongod·mosquitto 로그에 maxsize/maxage + epoch초 접미사 dateformat 추가로 같은 날 다회 회전 시 파일명 충돌·폭주 방어.
